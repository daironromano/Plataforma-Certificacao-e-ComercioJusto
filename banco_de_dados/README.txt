Protocolo "Onboarding" da Equipe

Passo A: Criar o banco de dados localmente
 - Motivo: o Git não salva o banco de dados MySQL, apenas o código em pyhton. 

1. Instalar MySQL no seu PC.
2. Abrir o workbench
3. Rodar o script 'banco_de_dados.sql'

Passo B: Configurar a Senha (settings.py)
 - Motivo: a senha do seu bd local é diferente da senha do coleguinhs

1. Abrir o arquivo settings.py
2. Procurar por 'USER' e 'PASSWORD'
3. Alterar para o seu usuario e sua senha

OBS: Não esquecer de instalar as bibliotecas.
